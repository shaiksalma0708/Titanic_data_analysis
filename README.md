# Titanic Data Analysis 🚢

This project involves performing Exploratory Data Analysis (EDA) on the Titanic dataset using Python in Visual Studio Code.

## 📌 Objective
To analyze passenger data and identify patterns and factors that influenced survival rates.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## 📊 Key Analysis Performed
- Data cleaning and preprocessing (handling missing values)
- Exploratory Data Analysis (EDA)
- Visualization of survival rates based on:
  - Gender
  - Age
  - Passenger class
  - Fare
- Correlation analysis between features

## 📈 Key Insights
- Female passengers had a higher survival rate than males
- Passengers in higher classes (1st class) had better survival chances
- Children had relatively higher survival rates
- Fare and class showed strong correlation with survival

## 📁 Project Structure
- `titanic_analysis.ipynb / .py` – Main analysis file
- `dataset.csv` – Titanic dataset
- `README.md` – Project documentation

## 🚀 Conclusion
This analysis highlights how different factors such as gender, class, and age impacted survival during the Titanic disaster.
